Este projeto é um aplicativo mobile desenvolvido para apoiar o grupo Maya RPG (Reeducação Postural Global), trazendo mais praticidade ao processo de cadastro de pacientes. Por meio dele, é possível registrar os dados de cada paciente de forma organizada e atribuir atividades personalizadas para serem realizadas em casa, permitindo que a equipe Maya faça o acompanhamento e as publicações de forma manual e individualizada para cada pessoa atendida.

A pasta raiz contém arquivos e pastas que seguem a seguinte estrutura:

<b>executáveis</b>: O arquivo APK do aplicativo está nesta pasta.

<b>imagens</b>: Imagens do sistema.

<b>src</b>: Pasta que contém o código fonte, dividido entre Backend e Frontend.

## 🛠 Instalação

<b>Android:</b>

Faça o download do arquivo <b>app.apk</b> na pasta executáveis diretamente no seu celular.<br>
Execute o APK e siga as instruções do seu dispositivo para concluir a instalação.

> Não é necessária nenhuma configuração adicional após a instalação. Basta abrir o aplicativo e utilizá-lo normalmente.

## 💻 Configuração para Desenvolvimento

Para rodar e testar o projeto localmente, siga as instruções abaixo para cada parte da aplicação.

<b>Backend:</b>

O backend está em produção e pode ser acessado diretamente pelo seguinte endereço:

```sh
https://sua-api-em-producao.com
```

<b>Frontend (Android):</b>

Para rodar o frontend em ambiente de desenvolvimento, você precisará das seguintes ferramentas:

- <a href="https://developer.android.com/studio">Android Studio</a>
- JDK 21
- Gradle (gerenciado automaticamente pelo Android Studio)

```sh
# 1. Clone o repositório do projeto diretamente no Android Studio:
#    File > New > Project from Version Control > cole a URL do repositório

# 2. Aguarde o Android Studio sincronizar as dependências via Gradle

# 3. Crie ou baixe um emulador Android pelo AVD Manager:
#    Tools > Device Manager > Create Device

# 4. Execute a configuração "app" clicando no botão Run (▶) ou com Shift+F10
#    para iniciar o teste em tempo real no emulador
```
